# BorrowBox - College Marketplace

BorrowBox is a student-to-student marketplace for borrowing and lending textbooks, electronics, and gear within your college campus.

## Features

- **Marketplace**: Browse items available for rent or sale.
- **Secure Login**: Student email verification required.
- **Premium UI**: Modern, responsive design with glassmorphism aesthetics.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Modern CSS Variables & Flexbox/Grid)
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `src/components`: UI components like Navbar, Buttons.
- `src/pages`: Page components (Home, Login, Signup).
- `src/index.css`: Global styles and design tokens.
